# 会社ポータル

これは実際に業務で制作した社内ポータルを、デモ用に一部抜粋、改変したものです。<br>

## 主な機能

- **PC管理**<br>
機材の貸出・所持者・返却状況をリアルタイムで追跡。資産管理の効率化と紛失防止に貢献。

- **メンバー一覧**<br>
社員のスキル・得意分野・興味を可視化し、プロジェクトアサインやメンター探しを効率化。MBTI診断結果やSlackID、自己紹介も登録でき、チームビルディングに活用可能。

- **図書管理**<br>
社内蔵書の貸出状況を一元管理し、新規購入リクエストもサイト内で完結。無駄な口頭確認やメールのやり取りを削減できる。レビュー・いいね・おすすめ機能により、読書を通じた社員間の交流促進にも貢献。

- **ヒヤリハット**<br>
インシデントの発生事実・原因・対策を体系的に記録し、改善効果（Before/After）を可視化。再発防止と安全意識の向上に寄与。

- **イベント履歴**<br>
社内イベントの開催記録をアルバムURL付きで管理。参加メンバーや資料を残すことで、組織文化の醸成と活動の振り返りをサポート。

### メリット
情報の一元化により業務効率が向上し、スプレッドシート連携で誰でも簡単にデータ更新が可能。ブラウザだけで動作するため、特別なインフラは不要。
